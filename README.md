# AI Video Assistant

An AI-powered Video Assistant that:

- Accepts YouTube videos or uploaded audio/video files
- Extracts and transcribes audio using Whisper
- Creates vector embeddings using ChromaDB
- Uses RAG (Retrieval Augmented Generation)
- Lets users ask questions about the video content
- Built with Streamlit, LangChain, ChromaDB, Whisper, and Mistral AI

## Features

- YouTube video processing
- Audio transcription
- Semantic search
- Question answering
- Vector database storage

## Tech Stack

- Python
- Streamlit
- Whisper
- LangChain
- ChromaDB
- Mistral AI
